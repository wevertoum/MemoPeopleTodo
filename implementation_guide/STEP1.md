# STEP 1 — Bootstrap (Next.js + Tailwind + TypeScript)

## Objetivo
Projeto web configurado com App Router, Tailwind e TypeScript, pronto para variáveis de ambiente do Firebase.

## Checklist

- [ ] Next.js 15+ com App Router
- [ ] TypeScript estrito
- [ ] Tailwind CSS v4 ou v3 (conforme template)
- [ ] ESLint
- [ ] Estrutura de pastas: `app/`, `components/`, `lib/`, `hooks/`
- [ ] Arquivo `.env.local.example` com chaves do Firebase (sem valores secretos)

## Critérios de pronto

- `npm run dev` sobe sem erros.
- Página inicial renderiza layout base.
- `README.md` na raiz do app descreve como copiar `.env.local.example` → `.env.local`.
